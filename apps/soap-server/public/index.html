1º Trabalho Prático da unidade curricular Desenvolvimento de Aplicações Web — implementação de um serviço SOAP em Node.js acompanhado por cliente CLI, proxy REST e landing page de apresentação.

Como executar

  1. Instale dependências na raiz do monorepositório: pnpm install.
  2. Inicie o servidor SOAP: pnpm --filter soap-server dev.
  3. Opcional: arranque o proxy REST para consumo via HTTP/JSON: pnpm --filter soap-proxy dev.
  4. Teste operações no cliente CLI: pnpm --filter soap-client start add 7 5.
Ver WSDL Repositório no GitHub Universidade do Algarve

Operações disponíveis

SOAP

<soap:Envelope>
  <soap:Body>
    <add>
      <a>7</a>
      <b>5</b>
    </add>
  </soap:Body>
</soap:Envelope>

Consuma no endpoint /wsdl com base no WSDL fornecido.

REST (proxy)

POST http://localhost:3001/add
Content-Type: application/json
{
  "a": 7,
  "b": 5
}

Resposta: { "result": 12 }

Objetivos de aprendizagem

  • Construir e publicar um serviço SOAP em Node.js com WSDL próprio.
  • Validar dados de entrada e devolver faults educativos sempre que necessário.
  • Implementar um cliente CLI e um proxy REST que reutilizam o serviço SOAP.
  • Documentar e apresentar o trabalho com identidade académica da UAlg.
