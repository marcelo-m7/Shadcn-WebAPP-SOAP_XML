SOAP Calculator — UAlg LESTI 2025/26

1º Trabalho Prático de Desenvolvimento de Aplicações Web
Autor: Marcelo Santos (a79433)

Contexto Académico

Este projeto concretiza o 1º Trabalho Prático da unidade curricular Desenvolvimento de Aplicações Web do curso LESTI — Licenciatura em Engenharia de Sistemas e Tecnologias de Informação, Universidade do Algarve (ano letivo 2025/26). O objetivo é implementar um serviço SOAP em Node.js que exponha operações aritméticas básicas, exercitando competências de integração de serviços e compreensão de protocolos orientados a mensagens.

Componentes Principais

  • SOAP Server: disponibiliza operações aritméticas via WSDL e serve esta página informativa.
  • SOAP Client CLI: permite invocar operações a partir da linha de comandos para fins de teste académico.
  • REST Proxy: expõe o serviço SOAP através de endpoints JSON, facilitando integrações modernas.
  • Shared Packages: partilham o WSDL e validadores de entrada entre aplicações.

Ligação Rápida

  • WSDL: http://localhost:3000/wsdl?wsdl
  • Repositório GitHub: github.com/marcelo-m7/UAlg-SOAP-TP1WEB
  • Exemplos XML: /examples no repositório

Exemplos de Requests SOAP

Invocar a operação add:

<soap:Envelope xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/">
  <soap:Body>
    <add>
      <a>7</a>
      <b>5</b>
    </add>
  </soap:Body>
</soap:Envelope>

Resposta esperada:

<soap:Envelope ...>
  <soap:Body>
    <addResponse>
      <result>12</result>
    </addResponse>
  </soap:Body>
</soap:Envelope>

Como testar rapidamente

  1. Instalar dependências: pnpm install
  2. Arrancar o servidor SOAP: pnpm --filter soap-server dev
  3. Executar o cliente: pnpm --filter soap-client start add 7 5
  4. Invocar via REST Proxy (POST JSON): curl -X POST http://localhost:3001/add -H "Content-Type: application/json" -d '{"a":7,"b":5}'
