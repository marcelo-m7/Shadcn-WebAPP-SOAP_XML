# Frontend Landing Page — UAlg SOAP TP1

Esta pasta documenta a landing page académica apresentada na raiz do servidor SOAP.

- **Localização do código fonte:** `apps/soap-server/public/index.html`
- **Objetivo:** disponibilizar um ponto de apresentação para o Trabalho Prático 1 da UC Desenvolvimento de Aplicações Web (LESTI 2025/26).
- **Como visualizar:**
  1. Iniciar o servidor SOAP com `pnpm --filter soap-server dev`.
  2. Abrir o navegador em [http://localhost:3000](http://localhost:3000) (ou executar `pnpm --filter soap-server open:frontend` em sistemas macOS/Linux com o comando `open`).
  3. Explorar os links para o WSDL, repositório GitHub e exemplos de requests.

> Nota académica: Esta interface tem natureza informativa/educativa e acompanha o relatório técnico do TP1, reforçando a autoria de Marcelo Santos (a79433) e o enquadramento na Universidade do Algarve.
